## Quick setup

```bash
git clone https://github.com/kddige/openclaw-fleet.git
cd openclaw-fleet
proto use && bun install
moon run fleet:dev
```

## Enforced rules

ESLint catches common mistakes at lint time. Every rule has a clear error message.

| Rule | What it catches |
|---|---|
| `from 'zod'` banned | Must use `zod/v4` |
| `from 'clsx'` banned | Must use `cn()` from `@/lib/utils` |
| `React.FC` banned | Use plain function components |
| `console.*` banned in electron/ | Use `createDebugLogger` |
| Query hooks banned in src/hooks/ | Use `useQuery` directly in components |

## PR process

1. Branch off `main`
2. One feature or fix per PR
3. Pre-commit hook runs lint + typecheck automatically
4. `moon ci` must pass
