## Download

Grab the latest release from GitHub — pick the installer for your platform:

[**GitHub Releases**](https://github.com/kddige/openclaw-swarm/releases) — `.dmg` for macOS (Apple Silicon or Intel), `-Setup.exe` for Windows.

## macOS

1. Open the downloaded `.dmg` file
2. Drag **OpenClaw Swarm** into your Applications folder
3. Launch from Applications or Spotlight

<Callout type="warn" title="App is not code-signed">
  macOS may block the app with **"OpenClaw Swarm is damaged and can't be opened"** or an unidentified developer warning.

To fix this, run the following in Terminal:

```bash
xattr -cr "/Applications/OpenClaw Swarm.app"
```

Then open the app normally. You may also need to go to **System Settings → Privacy & Security** and click **Open Anyway**.

</Callout>

## Windows

1. Run the downloaded `-Setup.exe` installer
2. If Windows SmartScreen shows a warning, click **More info** → **Run anyway**
3. Launch OpenClaw Swarm from the Start menu

## First launch

When you open OpenClaw Swarm for the first time, you'll see an empty gateway list. The app generates a unique **device identity** (Ed25519 keypair) stored securely via your OS keychain (macOS Keychain / Windows DPAPI through Electron's safeStorage).

This identity authenticates you with OpenClaw Gateway instances — no passwords needed.
