---
title: Sessions
description: Monitor and inspect agent sessions
---

Sessions represent active connections between AI agents and an OpenClaw Gateway. Fleet provides a live view of all sessions across your fleet.

## Session list

The gateway detail view shows all active and recent sessions. Each session displays:

- Session key (monospace)
- Agent identity
- Duration and token usage
- Cost breakdown

## Session detail

Click any session to see its full details including:

- Chat history (for chat-type sessions)
- Token usage statistics
- Session metadata from the gateway
